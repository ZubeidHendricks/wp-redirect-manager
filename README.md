# Redirect Manager

Create and manage 301/302/410 redirects from one simple screen, and catch 404s
before visitors hit them. Part of the
[Zub Plugin Factory](https://github.com/ZubeidHendricks/wp-plugin-factory).

## What it does

Add redirect rules, one per line — `/old | /new | 301`. Supports 301/302/307
redirects, 410 "Gone", and relative or absolute targets. Runs in PHP (no
`.htaccess` editing) and logs your top 50 most-hit 404s so you know what to fix.

## Settings

**Settings → Redirect Manager** — rules textarea, 404 logging, auto slug
redirect (Pro).
